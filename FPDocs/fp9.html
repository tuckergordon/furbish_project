Progress on Goals:

The focus of this week was getting everything connected: the search dialog now talks to the map which talks to the inspector. They're all connected. Here is a more specific description of the connections that we built:

  • Connected the search dialog with the map
    • Now if you select a flora, the map will update to reflect the locations of the samples for that flora. If there are no previous entries for a town, a dot will appear there. If there's already a dot at that location, it will grow in size. The dot scale is always updating, so dots don't get too big or small. This means that if you only have one flora selected, a town will have the same size as the largest town when all are selected. It's all relative to the others. We felt as though this made it easier for the user to process the data being displayed.
    • This also works for the Select All button
    • We also implmented deselection. If you click on either a highlighted (green) result in the search results list (on the left), or a flora in the selected flora list (on the right), it will remove all entries of the flora from the map. This means that dots will get smaller, and some might even disappear.
    • The same works for the Deselect All Button
  • Connected the map with the town inspector
    • Now if you click a town dot, that town and all of the entries (which have been selected in the search dialog) will appear in the inspector. This displays the town name, and information for every flora entry there, such as its names, the year of entry, and the volume and page(s) where it can be found in the collection.
  • Connected all three
    • Let's say you have 1 flora selected, which has an entry in Brunswick. You click on the Brunswick dot, and Brunswick pops up in the inspector, and contains that one entry. Now, let's say you select another flora, which also has a Brunswick entry. Not only will the dot adjust it's size, but now, as you still have Brunswick up in the inspector, that new flora entry will appear in the inspector.
    • Similarly, if you deselect a flora, it will remove all entries of that flora from the inspector (if there are any). If this leaves the inspector empty (i.e. there are no more flora entries for that town to display), the inspector will clear and reset so that it's not displaying any town (and that town's dot disappears from the map).
  • Made presentation
