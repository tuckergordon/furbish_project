Most Immediate Goals for the Following Week:

  • Reading in and parsing data
    • As this is a large data set, we are going to need to write a script that will parse the data and organize it in a format that we can use during the project. We are thinking of creating both flower and town structs to store the data, so during the parsing process, we will be creating a new instance of a flower struct whenever we encounter a new species of flora and a new instance of a town struct when we encounter a new town. To the flower struct, we will append:
      • The volume it was found in, and on what page it was found.
      • The scientific name of the flora.
      • The revised scientific name of the flora.
      • The AH scientific name.
      • The AH common name.
      • The towns in which this species was found.
      • The number of instances of this flora was found.
      • When the flora instances were found.
      To the town struct, we will append:
      • The name of the town
      • The flora that were found in that town
      • The number of each of the flora that were found in that town
      • The volume the flora were found in, and on what pages they were found.
      • When the flora instances were found.
  • Communicating plans/ideas for visualizations with Kat
    • As we are creating a visualization for a specific person, we want to contact this person again and discuss what they would like out of this visualization so that we can tailor our visualization to suit their needs.
